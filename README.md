# Math Dojo

## Introduction

A timed maths game desgined using HTML, SCSS, CSS and Javascript and implemented with a class using a mobile-first approach.

The game starts off on the home screen where you will be asked to select a level and a time. Once you have selected one from each option, pressing the start button will take you to the game screen.
